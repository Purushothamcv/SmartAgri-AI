## ğŸ› Troubleshooting

### "npm not found"
**Install Node.js:** https://nodejs.org/

### Port 3000 in use
**Change port** in `vite.config.js`:
```js
server: { port: 3001 }
```

### Map not loading
**Check:**
- Internet connection
- Leaflet CSS in `index.html`
- Browser console errors

### API errors
**Verify:**
- Backend is running
- `.env` file exists
- API URL is correct
- CORS is enabled on backend
